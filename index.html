بيانات حية وحساب شخصي

  1. يتم جلب المباريات تلقائياً من TheSportsDB API دون أي إعداد إضافي.
  2. أنشئ حساباً مجانياً لحفظ اسمك، وستحصل تلقائياً على توكن مشاهدة صالح لمدة 4 ساعات لفتح المشغل.
  3. لإضافة بث رسمي مرخّص، عدّل streams داخل config.js (يدعم embed وhls وvideo وexternal).
  4. نوع hls يشغّل تقنية Adaptive Bitrate عبر hls.js لتقليل التقطيع في البث المباشر بجودة عالية.
