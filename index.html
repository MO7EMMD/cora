إعداد سريع

كيف تربط الموقع مع API ومصادر البث؟

  1. افتح ملف config.js وضع مفتاح مزوّد بيانات المباريات.
  2. اختر نوع المزود: api-football أو thesportsdb.
  3. أضف روابط البث الرسمية داخل streams لكل مباراة أو لكل فريقين.
  4. إذا كان مزود البث يمنع التضمين، اجعل النوع external لفتح الموقع الرسمي.
