# كأس العالم مباشر

موقع عربي بسيط واحترافي لمتابعة مباريات كأس العالم، مبني بالكامل بـ HTML/CSS/JS بدون أي إطار عمل، ومنشور مجاناً عبر GitHub Pages.

**الموقع المباشر:** https://mo7emmd.github.io/cora/

## المزايا

- **بيانات مباريات حقيقية** من [TheSportsDB](https://www.thesportsdb.com/api.php) (يدعم أيضاً API-Football كبديل).
- **إنشاء حساب وتسجيل دخول** (محلي داخل المتصفح عبر `localStorage`، مناسب لموقع ثابت بدون خادم).
- **نظام توكن مشاهدة** يُصدر تلقائياً عند تسجيل الدخول، صالح 4 ساعات، ويُشترط لفتح المشغّل.
- **تشغيل HLS متكيّف الجودة** عبر [hls.js](https://github.com/video-dev/hls.js) لتقليل التقطيع في البث المباشر.
- **أخبار كأس العالم (اختياري)** عبر SerpApi، مع وسيط آمن يحمي مفتاح API — انظر القسم أدناه.

## التشغيل محلياً

```bash
cd files/worldcup
python -m http.server 4173
# ثم افتح http://localhost:4173
```

## إعداد مزوّد البيانات والبث

كل الإعدادات في `config.js`:

- `dataProvider`: مصدر بيانات المباريات (يعمل افتراضياً بمفتاح TheSportsDB التجريبي `"3"`).
- `streams`: خريطة `matchId → مصدر البث`. الأنواع المدعومة: `embed` (iframe مرخّص)، `hls` (رابط m3u8 مرخّص مع تشغيل متكيّف)، `video` (رابط فيديو مباشر)، `external` (رابط خارجي يفتح في المنصة الرسمية).

⚠️ **مهم:** أضف فقط روابط بث تملك ترخيصاً قانونياً لاستخدامها. لا يقدّم هذا المشروع أو يوزّع أي بث غير مرخّص.

## أخبار كأس العالم عبر SerpApi (اختياري)

الموقع يدعم عرض أخبار حقيقية عبر [SerpApi](https://serpapi.com) (واجهة نتائج بحث Google)، لكن هذه الميزة **معطّلة افتراضياً** لأن SerpApi:

1. لا يدعم الاستدعاء المباشر من كود المتصفح (لا يوجد CORS).
2. يتطلب مفتاح API سرّي — ووضعه داخل ملفات موقع ثابت يعني أن أي زائر يمكنه سرقته من "عرض المصدر".

لذلك تم بناء الميزة عبر **وسيط خادم صغير** (`serverless/serpapi-proxy.js`) ينشره صاحب الموقع بنفسه مجاناً على Cloudflare Workers، ويحتفظ بمفتاح SerpApi كسرّ على الخادم فقط.

### خطوات التفعيل

1. أنشئ حساب SerpApi مجانياً واحصل على مفتاحك من https://serpapi.com/manage-api-key
2. انشر `serverless/serpapi-proxy.js` على Cloudflare Workers (الخطوات التفصيلية مكتوبة داخل الملف نفسه كتعليقات).
3. أضف مفتاح `SERPAPI_KEY` كسرّ (Secret) في إعدادات الـ Worker.
4. انسخ رابط الـ Worker النهائي وضعه في `config.js`:
   ```js
   newsSearch: {
     enabled: true,
     proxyUrl: "https://serpapi-proxy.YOUR-SUBDOMAIN.workers.dev",
     query: "FIFA World Cup 2026 news"
   }
   ```
5. احفظ وارفع التعديل — سيظهر قسم "أخبار كأس العالم" تلقائياً في الصفحة.

## الملفات الرئيسية

| الملف | الوظيفة |
|---|---|
| `index.html` | هيكل الصفحة الرئيسية |
| `styles.css` | التصميم الكامل (سمة داكنة، دعم RTL) |
| `config.js` | كل الإعدادات القابلة للتعديل (مزوّد البيانات، البث، الأخبار) |
| `auth.js` | نظام الحسابات وتوكن المشاهدة |
| `app.js` | منطق جلب المباريات وعرضها وتشغيل البث |
| `serverless/serpapi-proxy.js` | وسيط آمن اختياري لميزة الأخبار |
